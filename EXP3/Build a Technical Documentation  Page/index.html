Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

Python's philosophy emphasizes code readability with the use of significant indentation. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

Python is dynamically typed and garbage-collected, supporting multiple programming paradigms including structured, object-oriented, and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.

Key Features of Python:

  • Easy to learn and use with clear, readable syntax
  • Extensive standard library and third-party packages
  • Cross-platform compatibility
  • Supports multiple programming paradigms
  • Large and active community
Getting Started

To begin programming in Python, you first need to install Python on your system. Python can be downloaded from the official Python website at python.org. The installation process varies depending on your operating system.

After installation, you can verify that Python is correctly installed by opening a terminal or command prompt and typing the following command:

python --version

This command will display the installed Python version. To start the Python interactive shell, simply type python or python3 in your terminal.

Your first Python program is traditionally the "Hello, World!" program. Here's how you write it:

print("Hello, World!")

Python files use the .py extension. You can create a file named hello.py with the above code and run it using the command python hello.py.

Variables and Data Types

In Python, variables are created when you assign a value to them. Unlike many other programming languages, Python does not require explicit declaration of variables or their types. The type is inferred automatically based on the value assigned.

Python has several built-in data types, which can be categorized into the following groups:

  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Text Type: str
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool

Here are examples of variable assignments with different data types:

# Integer x = 10 # Float y = 3.14 # String name = "Python" # Boolean is_active = True # List numbers = [1, 2, 3, 4, 5] # Dictionary person = {"name": "Alice", "age": 30}

Python variables are dynamically typed, meaning you can reassign a variable to a different type at any time. Variable names must start with a letter or underscore and can contain letters, numbers, and underscores.

Control Flow

Control flow statements in Python allow you to control the execution of your code based on certain conditions. The primary control flow structures include if statements, for loops, and while loops.

The if statement is used for conditional execution. It evaluates a condition and executes a block of code only if the condition is true:

age = 18 if age >= 18: print("You are an adult") elif age >= 13: print("You are a teenager") else: print("You are a child")

Python uses indentation to define code blocks instead of curly braces or keywords. This is a fundamental aspect of Python's syntax and enforces code readability.

The for loop is used to iterate over a sequence (such as a list, tuple, string, or range):

fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(f"I like {fruit}")

The while loop repeatedly executes a block of code as long as a condition remains true. It's important to ensure that the condition eventually becomes false to avoid infinite loops.

Functions

Functions are reusable blocks of code that perform a specific task. In Python, functions are defined using the def keyword followed by the function name and parentheses containing any parameters.

Functions help organize code into logical sections, promote code reuse, and make programs easier to understand and maintain. Here's a basic function example:

def greet(name): """This function greets the person passed in as a parameter""" return f"Hello, {name}!" # Calling the function message = greet("Alice") print(message) # Output: Hello, Alice!

Functions can accept multiple parameters and can return values using the return statement. If no return statement is specified, the function returns None by default.

Python also supports default parameter values, keyword arguments, and variable-length arguments. Here's an example with default parameters:

def calculate_area(length, width=10): """Calculate area with default width""" return length * width area1 = calculate_area(5) # Uses default width: 50 area2 = calculate_area(5, 20) # Uses provided width: 100

Functions in Python are first-class objects, meaning they can be assigned to variables, passed as arguments to other functions, and returned from functions. This enables powerful programming patterns like decorators and higher-order functions.

Object Oriented Programming

Object-Oriented Programming (OOP) is a programming paradigm that uses objects and classes to structure code. Python fully supports OOP principles including encapsulation, inheritance, and polymorphism.

A class is a blueprint for creating objects. It defines attributes (data) and methods (functions) that the objects created from the class will have. Here's how to define a simple class:

class Dog: """A simple Dog class""" def __init__(self, name, age): """Initialize name and age attributes""" self.name = name self.age = age def bark(self): """Simulate a dog barking""" return f"{self.name} says Woof!" # Creating an instance my_dog = Dog("Buddy", 3) print(my_dog.bark()) # Output: Buddy says Woof!

The __init__ method is a special method called a constructor. It's automatically called when a new instance of the class is created. The self parameter refers to the instance being created and must be the first parameter of any method in the class.

Inheritance allows a class to inherit attributes and methods from another class. This promotes code reuse and establishes a relationship between classes:

class Animal: def __init__(self, name): self.name = name def speak(self): pass class Cat(Animal): def speak(self): return f"{self.name} says Meow!" my_cat = Cat("Whiskers") print(my_cat.speak()) # Output: Whiskers says Meow!

OOP in Python provides a powerful way to structure your programs, making them more modular, scalable, and easier to maintain. Understanding classes and objects is essential for working with many Python frameworks and libraries.
